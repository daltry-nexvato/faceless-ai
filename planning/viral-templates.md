# Viral Script Templates & Retention Science

## Purpose
Built-in library of proven script frameworks that the AI uses as skeletons when generating scripts. Users pick a format, AI fills it with their topic + niche + style.

## Long-Form Templates (YouTube)

### Template 1: Educational/How-To
```
[0-5s]   HOOK: Shocking stat or contrarian claim
[5-15s]  PROMISE: "In this video, you'll learn [specific outcome]"
[15-30s] CREDIBILITY: Why this source is trustworthy
[30s-2m] PROBLEM: Define and agitate (PAS framework)
[2m]     OPEN LOOP: "But before the best tip..."
[2-4m]   POINT 1 + MINI-PAYOFF
[4m]     PATTERN INTERRUPT (visual change, B-roll, tone shift)
[4-6m]   POINT 2 + MINI-PAYOFF
[6m]     RE-HOOK: "Now here's where it gets interesting..."
[6-8m]   POINT 3 + BIGGEST REVELATION
[8-9m]   SUMMARY/TRANSFORMATION
[9-10m]  CTA: Subscribe + next video tease
```
Best for: tutorials, advice, guides. Optimal length: 8-15 minutes.

### Template 2: Listicle/Countdown
```
[0-5s]   HOOK: "X [things/mistakes/secrets] that [outcome]"
[5-15s]  TEASE: Show #1 briefly
[15-30s] CONTEXT: Why this list matters
[Each item] SETUP → REVEAL → MINI-PAYOFF
[Between items] Pattern interrupt every 2-3 items
[50% mark] Re-hook + engagement prompt
[Final item] Biggest payoff, most surprising
[End]    CTA + open loop to next video
```
Best for: faceless channels, compilations. Optimal length: 10-20 minutes.

### Template 3: Documentary/Story
```
[0-5s]   COLD OPEN: Most dramatic moment from the story
[5-15s]  QUESTION: "How did [subject] go from [A] to [B]?"
[15-60s] BEGINNING: Set the scene, introduce characters
[1-3m]   RISING ACTION: Things start going wrong/interesting
[3m]     OPEN LOOP: Tease the climax
[3-6m]   COMPLICATIONS: Escalating stakes
[6m]     PATTERN INTERRUPT: New perspective or revelation
[6-8m]   CLIMAX: The peak moment
[8-10m]  RESOLUTION: What happened after
[10-12m] LESSONS/TAKEAWAYS + CTA
```
Best for: true crime, history, mystery, biopics. Optimal length: 12-25 minutes.

### Template 4: Comparison/Versus
```
[0-5s]   HOOK: "[A] vs [B] — which one actually wins?"
[5-15s]  CRITERIA: "We'll compare on [3-5 factors]"
[Body]   Alternating points for each side with mini-verdicts
[75%]    Surprise twist or unexpected finding
[End]    Final verdict + reasoning + CTA
```
Best for: tech reviews, product comparisons. Optimal length: 8-15 minutes.

### Template 5: Explainer/Deep Dive
```
[0-5s]   HOOK: Counterintuitive fact or question
[5-30s]  CONTEXT: Why this matters right now
[30s-2m] FOUNDATION: Basic concepts (quick, don't bore experts)
[2-5m]   LAYER 1: First level of depth
[5m]     PATTERN INTERRUPT + RE-HOOK
[5-8m]   LAYER 2: Deeper implications
[8-10m]  LAYER 3: What most people miss
[10-12m] CONCLUSION: Key takeaways + CTA
```
Best for: science, finance, philosophy. Optimal length: 10-18 minutes.

## Short-Form Templates (Shorts/Reels/TikTok)

### Template 6: Quick Tip (15-30s)
```
[0-1s]   Visual pattern interrupt
[1-3s]   One curiosity-gap sentence
[3-10s]  Setup the problem
[10-25s] The solution/value
[25-30s] Payoff + loop back to hook
```

### Template 7: Seamless Loop (15-30s)
```
Last line of script = hook/first line
Creates infinite loop effect
200% retention when viewer watches twice
Structure: [Hook/End] → [Setup] → [Value] → [Connects back to Hook]
```

### Template 8: POV Format (15-45s)
```
[0-2s]   "POV: [relatable situation]"
[2-20s]  Visual storytelling showing the scenario
[20-30s] Unexpected twist or punchline
```

### Template 9: Mini-Story (30-60s)
```
[0-3s]   "That one time when [intriguing setup]..."
[3-20s]  Build tension quickly
[20-40s] Escalation
[40-50s] Climax/twist
[50-60s] Resolution + loop potential
```

### Template 10: Before/After (15-30s)
```
[0-2s]   Show the "before" state
[2-5s]   "Watch what happens..."
[5-20s]  Transformation process (compressed)
[20-30s] Reveal the "after" — maximum contrast
```

## Hook Library (Categorized)

### Curiosity/Secret
- "No one talks about this in [industry]..."
- "I probably shouldn't share this, but..."
- "The truth about [topic] that [authority figures] won't tell you"
- "What nobody tells you about [topic]"
- "Here's why [common belief] is completely wrong"

### Challenge/Contrarian
- "Stop doing [common thing]. Do this instead."
- "You're doing [common mistake], and it's costing you [pain]."
- "Everything you know about [topic] is wrong"
- "[Number]% of people get this wrong"

### Value Promise
- "In the next [time], I'll show you [specific result]"
- "This one trick [achieved specific measurable outcome]"
- "Here's the [topic] strategy that [specific result with number]"

### Shock/Stat
- "[Shocking statistic]. Here's what that means for you."
- "I just discovered something about [topic] that changes everything"
- "[Authority figure] just revealed [unexpected thing]"

### Question
- "What if [unexpected scenario]?"
- "Did you know [counterintuitive fact]?"
- "Why does [common thing] actually [unexpected outcome]?"

### Story
- "That one time when [intriguing setup]..."
- "I used to think [common belief]... but then this happened"
- "Day 1 of [challenge/experiment]..."

## Retention Rules (Baked Into Every Script)

### Timing Rules
- Pattern interrupts every 20-30 seconds
- Open loops at key drop-off points (25%, 50%, 75% mark)
- Mini-payoffs every 60-90 seconds
- Re-hooks at every section transition
- Hook must achieve 70%+ retention at 30-second mark

### Benchmarks
- Overall average retention: 23.7%
- Top performers: 50%+ retention
- Educational how-to content: 42.1% average (highest niche)
- 55%+ drop-off occurs within first minute if hook is weak

### Critical Drop-Off Points to Defend
| Timestamp | Risk | Defense |
|-----------|------|---------|
| 0-5s | First decision point | Strong hook with pattern interrupt |
| 5-15s | Steepest drop-off | Deliver on thumbnail/title promise |
| 25% mark | Casual viewers drop | Open loop or curiosity gap |
| 40-50% mark | Midpoint slump | Re-engagement spectacle |
| 75% mark | "Almost-done syndrome" | Tease remaining value |

### AI Narration Warning
Monotonous AI narration = 35% higher drop-off in first 45 seconds.
Mitigations built into voiceover generation:
- Vary pacing and emphasis
- Add music beds and sound effects
- Use "energy markers" in script (speed up, slow down, emphasize)
- Consider multiple voice styles within one video

## Thumbnail Generation Rules

### Design Rules
- Max 4 words of text (0-4 words = highest CTR)
- 1 main subject + 1 context element + bold colors
- Emotional expressions: surprise, disbelief, awe > neutral
- Before/after thumbnails = 35% higher CTR
- Rule of thirds composition
- Minimum 30% free space
- Text in upper portion (avoids timestamp overlay)
- Contrast ratio minimum 4.5:1
- Resolution: 1280x720, sRGB

### CTR Benchmarks
- Platform average: 2-10%
- Average channels: 3-4%
- Top creators: 5-10%
- Elite: 15%+

## Title Generation Rules

### Structure
- Optimal: 55-60 characters
- Most important words FIRST
- Two-part structure: intrigue + clarity
- Numbers increase CTR 20-30% (digit "7" particularly effective)

### Power Words by Category
- Curiosity: "Secret," "Hidden," "Shocking," "Untold"
- Urgency: "Now," "Fast," "Today," "Before It's Too Late"
- Authority: "Expert," "Best," "Proven," "Ultimate"

### Title-Thumbnail Synergy
Title and thumbnail must tell a cohesive story together. Thumbnail = visual hook. Title = verbal hook. Complement, don't duplicate.

## Shareability Triggers (What Makes Content Viral)

Top 5 sharing triggers:
1. **Awe** (25% of viral content) — "I have to show people this"
2. **Laughter** (17%) — "My friends would find this hilarious"
3. **Amusement** (15%) — "This reflects my taste"
4. **Identity Expression** — "This represents who I am"
5. **Social Currency** — "Sharing this makes me look knowledgeable"

High-arousal emotions (awe, excitement, anger) trigger sharing.
Low-arousal emotions (contentment, sadness) do NOT.

- **YouTube** — Built into Niche Finder (outlier detection, breakout channels)

- **X/Twitter** — Trending topics by location. $200/month Basic tier.
